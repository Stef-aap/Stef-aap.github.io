Robot 1

Movie 3:50 , 1.8 MB  september 2007

This demo shows a robot, with a left and right motor and a left and right whisker. The environment of the robot can easily be adapted. This may sound weird, because we have only one JAL-program for one PIC, but as the robot is a virtual device the simulation can contain an unlimited amount of robots, all controlled from the one JAL program.


Alarm Clock

Movie 1:10 , 2.5 MB  august 2007

This demo show an alarm clock (a real application), with all kinds of extra features, like a different alarm time each day, programmable from a RS232 line etc. The JAL code automatically creates the 7-Segment display and 2 virtual buttons and no further schematics is needed. The program uses all kinds of libraries e.g., strings / format / data-eeprom / serial RS232 / delays

This demo revealed the following problems with the translation in JALsPy:

  • boolean operators are handled as bitwise operators, resulting in wrong code
  • put procedures are not handled correctly
  • volatile not handled
  • if statement complete on 1 line


7-Segment Display

Movie: 2:10, 1 MB, august 2007

This demo shows a simple JAL program running in JALsPy and controlling a 7-Segment Display. As the 7-Segment display is created as a super virtual component, the demo shows how the necessary hardware is automatically created by just running the JAL-simulation. This is the first demo that fully runs in the simulator without any tricks.

This demo shows the first ideas of what the JALsPy simulator should be capable of. Although what you see is a real running Python program, but most of the things were hard coded, the JAL-file was manually translated into Python code.
