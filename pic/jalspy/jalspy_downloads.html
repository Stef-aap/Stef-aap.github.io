Requirements

JALsPy is developed in Python, with wxPython as the GUI. So it should run on almost any platform.

The following units are windows only and should be replaced:

  • system_support, not strictly needed depends on your hardware uploader (contains dummy for other OSs)



As far as we are aware, these are the minimal versions of the different components (it's adviced to install them in this order)

  • Python 2.4  ( 2.5 seems to work also )
  • wxPython 2.8 (2.8.4.0 ?), don't forget to view the wxPython demo, very beautiful and instructive  !!



Not strictly necessary, but as long as JALsPy is in the alpha stage, it's recommended to run JALsPy from an IDE, so errors can be easily traced. There are a number of IDE's ( some for a specific operating system, some working on all operating systems).

Personally I use

  • PyScripter (windows only), should be run with the standard remote engine
  • Rpyc interface to external machines
