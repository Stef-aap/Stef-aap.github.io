JALss, building full-DLL-device

  

  

 

DLL_TESTDPR.ZIP contains the following file

  • DLL_TESTDPR. (cfg,dll,dof,DPR,res) for Delphi-5, it could be necessary to change the locations of the library files in the project 
  • DLL_dev_test_DLL (PAS,dfm), this is the real device code 
  • Janshape3.zip (special graphical shapes from Jan Verhoeven,  these components     should be installed in Delphi 
  • msg_glob.pas, contains all messages and claims an unique message in the windows environement 
  • device_types.pas, contains all type definitions
