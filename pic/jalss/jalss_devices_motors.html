Stepper Motor 

At this moment the stepper motor device is only implemented as a 4-phase unipolar stepper motor (most common used).

The stepper motor (central in the picture on the right) has a reduction box (gear at the left of the stepper motor), a servo potmeter (left of the gear) output and max, mid and min switch outputs.

The reduced angle (after the reduction box) is represented both numerical and as a visual gauge.

Windings 1a and 1b represents the 2 windings (with center tap) on the same poles.

Windings 2a and 2b representw the windings on the other poles.

A good tutorial about stepper motors on line is written by Douglas W. Jones.

The motor-animation was taken from this tutorial.

Timegraphs

The second page displays motor-angle, reduced-angle and speed as a function of time. These graphs will only be drawn as this page is selected (to save computer time).

Motor-angle and reduced angle are represented modulo 360 degrees.

Speed is graphical displayed as the absolute speed between 0 and 300 steps/sec.

The numerical value will display also the direction.

 

  

  

Input Signals

The third page displays the signals aplied to the windings of the motor.

Every pixel in horizontal direction represents a 1msec sample.

If simulation is stopped, moving the mouse over the image, will generate a green vertical line for a better time comparison of the signals. In this case also a number will be shown in the lower-left corner, indicating relative time in msec.

 

  

  

Motor settings

Unipolar and number of windings is fixed at the moment.

Reduction should be a positive integer.

Start Position, is the starting angle of the reduced angle that will appear after a reset. By setting this value to -1, a random value between 0 and 360 degrees will be generated.

Missing is the percentage of steps, that the motor looses steps, so you test programs that are watched by servo-potmeter and/or min/max-switches (not yet implented).

Min, Max and Mid switches will give a signal when activated, the logical active level can be selected. For each switch you can specify the range (in degrees from 0 to 360) of the reduced angle that will activate the switch. 

Maybe it's good to add some hysteresis in the future.

 

  

  

PulsWidths

A limited number of pulses can be analysed numerical.

The first value indicates the stepsize:

-2 = fullstep backwards

-1 = halfstep backwards

+1 = halfstep forwards

+2 = fullstep forwards

The second value indicates the time from the previous transition.
