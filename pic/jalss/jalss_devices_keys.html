JALss, Keyboard & Switches

  

  

 

Matrix Keyboard 

The matrix keyboard can contain a maximum of 25 keys.

Within that limit any combination of rows and coluns can be made.

Multiple Keys can be generated.

The duration of the keypress can be set.

If bouncing is given a value other than 0, the key will be active for half the bouncing time, then it will be inactive during half the bouncing time and after that it will be active again for the rest of the duration time.

All matrix line are connected through a resistor with either Vpos or Ground.

 

  

  

DIP Switches 

A number of Switches, mainly intended to set status information.

A new instance of this device is automatic connected between Vpos and Ground, but you can change it in the normal way and connect the input signals to something else.

At the moment the resistor has no special value and will act as general pullup or pulldown element.

Each switch has a text label in front of it, which can be used fro identifying by the user.

When decreasing the number of outputs, the status of the removed switches is maintained but will not be updated when the input signals changes. Maybe in the future it's better not to allow to remove outputs that are used by other devices.

This device is always a slow device.
