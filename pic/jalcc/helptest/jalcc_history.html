Installation 

Download the EXE-file in the directory of your choice. 

to install, just run the setup file (clean installation, files are only placed in the directory of your choice, no changes of the registry or whatsoever)

to uninstall, run unins000.exe, then delete the directory (to remove files that were changed by the user)

In the complete installation package of JALcc, JAL and all it's related files are installed just as they are in the JAL package. Therefor Installing a new JAL version is very simple, just unpack / overwrite everything, starting in the correct directory.

 Copy + Include + Open, meant for automation of nicely including for instance PIC definition-file, 

Assuming the program directory and library directory are distinct, 

Writing a program that uses a PIC, let say 16F628, 

Selecting this menu-item and selecting 16F628_DEFS.JAL in the library directory, 

Will result in the following actions: 

0. if the cursor is in a line with an include statement, this line is removed 

1. a copy of 16F628_DEFS is made in the program directory 

2. the statement "include 16F628_DEFS" is made at the cursor position 

3. the copy is opened in the editor and gets focus 

By naming these kind of files "*_DEFS.JAL" they can be accessed very easy.
