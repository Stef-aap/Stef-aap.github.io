JALcc, another IDE

                                                                    

This program couldn't have been written (in the limited available time) without the people who made their creations available as freeware  or shareware components, thank you all !!

        

 

Document List

The document list, organizes documents, web-links, e-mail addresses, music etc. A search can be performed on either title or filename/URL.


  • each element can be linked to 2 different applications
  • each element can be identified with it's own font attributes and icon
  • generates complete document from a sub-directory
  • webgeneration / verification / correction / uploading from a subtree
  • full screen presentation from a subtree
  • organized audio playlist
  • full drag and drop capabilities

  



Minimized multi-drop target

based on tRichViewDroptarget

When the application is minimized, a transparent stay on top floating icon will stay on the desktop, that can be used as a multi-drop target.

Accepts content / files / URLs.

Dropping can be done in the document list / weekly drop document / general drop document.

 


Wysiwyg Editor

based on tRichView  with special thanks to Sergey Tkachenko, for his fast, excellent and persistent support. Also thanks to Carlo Kok for his RVhtmlParser.


The editor is a very simple wysiwyg editor, capable of handling HTML*, RTF, RVF and TXT files. The concept is specially designed to make a number of (reasonable looking) documents, all having an equal style, in a very easy way. To give an impression of the possibilities, this documentation and this website are completly made by the Wysiwyg Editor in JALcc


The editor also contains a spell checker, based on ISpell, which can handle both own dictionaries and the (older) M$ Office dictionaries.



Picture Editor


The wysiwyg editor has a build-in picture editor, with some simple tools for adding arrows, labels and free drawing.




Formula Editor


The wysiwyg editor has a build-in formula editor, based on MimeTex from John Forkosh Associates.




Code Editor  

based on SynEdit

                            

Suited for any language (special support for JAL), completely configurable by the user.

 

This editor is missing just one thing: code-folding.



Code Parsing


The Object Tree, shows a hierachical list of objects in the project.



Stack analysis


Analysis procedure depencies up and down, specially stack usage and shows statistics about each procedure used. Especially usefull for PICs, because of their limited stack size.



JALcc macros

JALcc macro's extends the possibilities of JAL, while all files maintain (almost) fully compatible with JAL.

For the moment only a number of predefinied macros are available.

In JALcc, normally you'll only will see the macro layout.



TI-83 Basic Editor

The CodeEditor has a full code colored highlighter, and besides all the standard features of the general CodeEditor, it has a number of special features adapted to TI-basic.

The organized tree on the left can be used to insert functions into the editor, and moreover each item in the tree is associated with a (small) textfile, which can be used by the user to store notes about certain functions.

The Statusbar displays the status of the editor, but all the colored panels in the statusbar also act as buttons which performs certain actions



TI-83 (Enhanced) BZC Editor

Just as TI-Basic, JALcc supports also "enhanced" BZC (written by Erik van 't Wout & Eugene Evans). BZC is a block structured language, with a very good human readable syntax, like Pascal, Algol or modern Basic, which is compiled directly into TI-assembler.



Calculator

Just a good calculator, with a clear overview of parameters, formulas and results, which are also automatically saved.

           

On the right a typical example, how to use the calculator, specific for calculations of the PIC-usart parameters. Of course this can easily extended to all kind of other calculations for timers etc. In this example, just fill in the Xtal frequency in T and the desired baudrate in B, and the optimal setting is calculated, with the realised accuracy.



Dot Matrix Generator


This tool is meant to create graphics and special characters for LCD devices and or moving LEDs, used in PICs, like the famous Bob Blicks propellor clock.



Icon Manager


The Icon Manager lets you retrieve, organize and edit icons.




Hex Editor

Based on mirkes  wonderfull hexeditor component, thanks to Markus Stephany.


A lot of beautiful features TMPHexEditor are deliberatly left out, to fullfill the KISS principle.


Besides attaching the Hex Editor explictly to a document, you can open each document in the document list in the Hex Editor, through the RM-menu.




XML Viewer / Editor

preliminary

The viewer can be used for all kind of tree organized data, like XML, ini-files, structured programs. Both treeviews and code-highlighters and CSS views are possible.





Vector Editor

based on Diagram Designer  , thanks to Michael Vinther.


Usefull for drawing flowcharts, statemachines, electronic circuits, ...

You can easily add new objects and import even graphics.

The treeview on the left shows the hierachical structure of the document, which can exists of more layers. For making presentations, there's a background layer which is equal for all drawings.

  


Statemachine CodeGenerator

based on Diagram Designer


Generate code from a graphical designed statemachine, for JAL, Delphi and ..

At the moment this part is quiet preliminary as I haven't done any real world applications in it.



Veroboard Design

based on Diagram Designer


Quiet preliminary and limited, I've only done 1 project with it, but it looks nice especially for youngsters.



Document Generator


JALcc can generate a report, including cover and contents-list, from a number of single documents. It's also possible to include dynamically generated documents by other programs (see dynamic pages). It even contains scripting features, but these are not revealed yet.

                  



Website generator / verifier / uploader


After you've set your website parameters once, you're just 2 clicks away from generating + verifying + uploading a complete website from a subtree of the documentlist.


Of course this website was generated with this tool.




Quick Lauch


The top of the document list links to a kind of desktop window, where you can place shortcuts to often used programs. Launching one of these programs, adds the program (with its settings) to the quick launch bar at the top.

                       

So normally the quick launch bar will hold your latest used programs. But each element of the quick launchbar can be locked.



Tabbed Webbrowser

momentary this browser is based on IE, thanks to Bill Gates. As soon as the Mozilla interface is stable enough, the user can select his own favorite.


This browser was develloped because management of tab-sets in Mozilla is still not adequat enough.



Audio Playlist

You can play music form the document list, and even use the document list as an audio playlist. Different audio players are available. Special thanks to Boian Mitov  for his AudioLab (Boian Mitov's C++ Builder site - Home) and his fast, excellent and persistent support



General Viewer

Presentation Manager


This viewer (capable to show a large scale of different documents) displays the selected item in the document list, so you can easily step through the document list.


This window is also used as presention manager, by displaying it full screen, with some extra control tools and a transparent watermark.



Image Organizer

based on VirtualTreeview  , thanks to Mike Lischke  and ImageMagick

Orginally designed to quickly get parts of an image collection of a camera into documentation and finally clearing the camera for next use.

So it just has some very basic tools, like rotate/flip/resize. But I find this tool so handy, I've plans to extend it a little bit in the future.



File Browser

based on VirtualTreeview  , thanks to Mike Lischke.

When manipulating a lot of files in windows, you'll find out how crappy M$ explorer is (even after increasing MRUbags in the registry to astronomuous numbers). With the good experiences of the image organizer, I decided to build an explorer on the same base.

This browser can be started from the tools menu and from the document list.


Time Series Viewer

  

This module is intended as a viewer (and maybe later on as an graphical analyzer) for time series of different programs. At this moment the following file formats are recognized: Spike2-CFS and Midac. Plans are there to add Spike2-SMR, Labview, Matlab, NIRS.



File Encryption

based on 7Zip,  thanks to Igor Pavlov.

Any file in the document list can be transparently encrypted by the strong AES-256 encryption method.



Quality FeedBack Agent

or in simple words: auto bug reporting,

based on MadExcept  thanks to Mathias Rauen.

If the generated bugreport is posted to me, I can in most cases trace the problem quiet easy.
