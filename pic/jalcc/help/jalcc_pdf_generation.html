PDF Generation

  

       MainIndex

  

  

Although this has nothing to do with JALcc, PDF generation (and viewing) is an important task, which gave me always a lot of trouble.

Even with the professional version of Adobe PDF-creator (V6), I had numerous problems:

- viewing 2 pages next to each other

- continuous complaining about corrupted files (no problems with other viewers)

- continuously want to update, and then updates crashes

- continuously all kind of other error messages

- waiting upto 30 seconds to start up

  

If you've ever seen (freeware programs) like GhostScript, GhostView, Foxit or PDFCreator, you never want to see Adobe again !!

  

First Install GhostScript

  ghostscript.com

  

Then Install a pdf viewer, some freeware programs:

  GSview

     or

  Foxit PDF Editor from Foxit Software

  

  

Adobe Reader 5.05 .. 5.5 are free and quiet fast and good.

 

  

  

And finally install  

  PDF-Creator

Redmon seems to be quiet good, but somewhat difficult to install.  RedMon - Redirection Port Monitor
