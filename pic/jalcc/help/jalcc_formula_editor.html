Introduction

With the button   you can insert a new formula.

After selecting a formula image, the right mouse menu gives the possibility to modify an existing formula.


       


(The formula is stored in html as the alternate text property of the image, with a recognizable markup).

All images are created as transparent gif-images.

Editor

The editor uses MimeTex (almost equal to the well known Latex) to create an image of the formula. See for extended information about MimeTex John Forkosh Associates.


       


On the left-top you see the editor field, in which you can edit the formula. New lines are automatically translated into the double slash needed for MimeTex.

On the left-bottom you see the image generated by MimeTex.The size can easily be controlled by the spin button at the right bottom (which will add an appropriate size string for MimeTex).



Templates

On the right there are a number of tabbed pages, the first page is a general help text, the other pages are so called template pages. Both the help page and the template pages can be fully customized by the user. Up to 9 template pages can be created (files in the program's root directory with the name "formula_templates_X.txt", where X is a number from 1 to 9) .

The help page is a normal plain text page and can be edited, by removing the readonly flag, by pressing the Edit button (at the bottom).

The template pages contains a number of templates, when you click on a template, the template is inserted in the formula editor at the place of the caret.

When a template page is selected, pressing the Edit button starts editing the template.


       


You can see 2 special characters in this text:

  • -   a minus sign will insert an horizontal line
  • |   a vertical bar will insert an vertical line and add a new column.
