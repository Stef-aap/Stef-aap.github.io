All sequence types are nestable (without any depth restriction), for list, tuple, dictionary, array:

    L3 = [ 2, 'hello', [ 45, 'aap', [ 2, 4 ] ] ]

    T3 = ( 2, 'hello', ( 45, 'aap', ( 2, 4 ) ) )

    D3 = { 'aap' : 1, coala: { 'beer': 2 } }

    A3 = array ( [ [ 1, 2 ] , [  4, 5 ] ] )

or even in every random combination / nesting:

    R3 = [ 2, 'hello', ( 45, 'aap', [ 2, 4 ], { 'aap' : 1, coala: { 'beer': 2 } } ) ]


List and Tupple are different in their mutability and therefore also their methods differ.


IMMUTABLE:

  • the variable can not be changed in-place
  • for simple variables (e.g. integer) passed to functions, modifications in a function body creates a new local variable, for complex variables  (e.g. tuple) an error will occure. This error can be prevented by making an explicit x.copy  or x.deepcopy.


MUTABLE:

  • the variable can be changed in-place
  • the variable is passed by reference to a function
