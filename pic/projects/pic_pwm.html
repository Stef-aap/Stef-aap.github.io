PWM with RC feedback

(work in progres)

  

Trace_duration  > Step Duration

TD > SD

length [samples] = sum (SD) + (TD-SD)

abs(height) [AD] = SD + sum(TD-SD+1)  -1

definition:  sum(A)  = 1 + 2 + 3 + ... + A =  A * (A+1) / 2

 

  

  

  

 

  

  

  

 

  

  

Curve_up_start  (1,5)   (3,6)   (5,3)  (10,9)

Curve_up_end   (1,5)   (3,6)   (5,3)

step_duration=1, trace_duration=5

TD > SD

length = sum(SD) + (TD-SD)  =  (1*2 / 2) + (5-1)  = 1 + 4 = 5

height = SD +  sum ( TD-SD+1) -1 = 1 + (5*6/2) - 1 = 15

step_duration=3, trace_duration=6

TD > SD

length = sum(SD) + (TD-SD)  =  (3 * ( 3 + 1 ) / 2) + (6-3)  = 6 + 3 = 9

height = SD +  sum ( TD-SD+1) -1 = 3 +  4*(4+1)/2  - 1 = 12

step_duration=5, trace_duration=3

TD <= SD

length = sum(SD) -sum(SD-TD) = sum(5) - sum(2) = (5*6/2) - (2*3/2) = 15-3 = 12

height = TD = 3
