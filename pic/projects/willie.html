Willie / RobbieBot

  

  
  

All scripting is stored in one file:  <application-path>\robbie_script.txt
The script file can contain more then 1 program, because an empty line serves as the end of a program.
A program (or part of it) is started, by putting the cursor on the line where the scrpt should start and then pressing ctrl-enter.
  
Scripting is not case-sensitive.
A line starting with a semi-collon ";" is interpreted as comment.  
The general command structure is  "<MOTOR>    <POSITION>"
For the moment <POSITION> is not translated into degrees, but just send as a raw value to RobbieBot. Hex values may be entered, by preceeding them by a dollar-sign "$".
  
D  =  draai
A  =  arm1
B  =  arm2
K  =  knijper ( 0 or 1)
W =  wait  (units of 100 msec, determined by getting a complete position line form RobbieBot)

Scripting is executed as fast as possible, but execution pauzes on one of the following conditions:

  • a command for a motor is given, while the previous command for that motor has not finished
  • a wait command is encountered
