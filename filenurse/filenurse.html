Synchronize without transport medium:

Create an ini-file, as a copy of an existing filenurse backup file. Now start FileNurse with as the first and only parameter the name (and path) of this copied file). Edit the synchro settings and "Save". Now this backup-inifile can be used to synchronize 2 computers.


In the backup-inifile, the following section will be added:


        [BackUp]

        other disk=\\Birgit\S\

        cb_insert_driveletter=0

        cb_synchro_relative=1

        cb_leave_equals=0




Windows Installation Changes:

You have to declare the first part of the path where the backup should be stored, in the programs main ini_file "FileNurse.ini" (located in the application's directory:


        [general]

        backup file=U:\schijf_image\amd_64_


You have to install the autoit program in the application's directory:


        DrvImagerXP_auto.exe


You have to install DrvImager (freeware)

When a change in the windows installation is detected, the program will ask a few questions and start the backup.

Introduction

FileNurse is a program that perfoms all kind of file and disk maintenance. It has the following features:

  • 2 panel browser
  • extended and fast file search
  • backup / synchronize function
  • ASCII file compare
  • special folder browser
  • logfile
