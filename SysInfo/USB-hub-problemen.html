You can't highlight and remove the Device Manager categories, and you

can only remove one device at a time, at which point the category

collapses again.

 

1. Click on + next to USB section. Select first item under it, Remove.

2. Repeat until the category itself disappears (meaning there are no

longer any devices listed there.)

 

Bu *sure* to do this in Safe Mode. It's just as safe in Normal Mode, but

in Normal Mode all you see is devices that have been loaded, whereas in

Safe Mode you see all installations. This is an important distinction,

since one of the main reasons for doing this procedure is to get rid of

"ghost" devices--devices that have been installed but are no longer

present, or devices which have been installed two or three times but

only one instance loads.

 

Really, except for the Video card, this procedure is pretty safe. Video

devices should not be removed this way except in extremis (and I

wouldn't recommend it for any novice.) Only other consideration is that

devices aren't usually stand-alone. You install many devices "onto"

other devices. This is where we get the terms "Parent" and "Child"

devices. More in-depth usage of this procedure involves remembering to

reinstall any Child device of any Parent device that you reinstall. Even

that isn't a hard rule, and the whole thing is pretty forgiving. Might

not fix your problem, but it seldom creates others.
