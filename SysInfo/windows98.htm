Installatie Vobis
 

Windows 98 installatie (ongeveer 30 minuten)

    2* startup floppy maken (i.v.m. slechte kwaliteit floppies)

    D ver leeg maken

    alles van C naar D kopieren (vergeet de ROOT niet), 1 file kan niet worden meegenomen c:\windows\...386... deze moet geexcluded worden anders stopt copy proces

    restart, wijzig BIOS door DEL, boot := A,C

    start with CD-ROM support  (F=CDROM1, G=CDROM2 (R/W)

    format C:

    sys C:

    restart

    F:

    setup, start pas na een aantal malen reboot, foutmelding " at least ... bytes nodig voor setup"  ???

    als gevraagd wordt, wederom een startup disk aanmaken

    er ontstaan omdat bepaalde files van C (komend van A?) niet goed uitgepakt kunnen worden, lijkt echter door te gaan

    restart, wijzig BIOS door DEL, boot := C,A


Drive Letters goed zetten 
    Drive letters goed zetten (zodat ruimte voor ZIP op E:)

        Control Panel / System / Device Manager / CDROM / ... / properties / ... instellen

            CDROM1 = F

            CDROM2 = G

    restart

Soundcard 
windows installeert automatisch een soundcard, die in de hardware devices met een geel vraagteken staat en ook niet  werkt
    Verwijder de door windows geinstalleerde componenten

    restart, windows constateert nu nieuwe hardware

    windows mag zelf zoeken, maar de officiele drivers installeren

        D:\CDROMs\Naar CD-ROM\Drivers en patches\Sound Boostar 16-32 3D\...

    restart en de soundcard lijkt perfect geinstalleerd
