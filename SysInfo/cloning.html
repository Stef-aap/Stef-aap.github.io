Boot.ini


Image maken


Start DrvImagerXp (er verschijnt een verklarende text in notepad dat het programma maskeert)

Selecteer de schijf die gekloond moet worden

Selecteer een directory en sla het image op zonder extensie.

Kies een grote voor de image bestanden (beste is 300MB)


"Save the drive partition to a partition image file" en volg de stappen


YES !!


Een Image van 8GB duurt ongeveer 6 minuten.

Daarna komt MSconfig met allerlei vragen, omdat je de instellingen hebt gewijzigd,

de uitgeschakelde programma's kunnen nu weer worden ingeschakeld.


Image restore


Start DrvImagerXp

Selecteer de schijf die gerestored moet worden

Open het image bestand (open de eerste)

Als de schijf geformatteerd moet worden zegt ie dat, volg de stappen:


(image size, # of files, zijn ingevuld en "Restore..." is zwart)



ALS restore andere SIZE

Als de schijf een andere grote heeft dan de oorspronkelijke, moeten de volgende stappen worden gedaan op de gekloonde schijf.


Geef de kloon een andere naam voor de duidelijkheid, het is misschien handig om ook de andere schijven een logische naam te geven.

scandisk draaien met "fix system errors".



Start Partition Magic

Resize: Maak de schijf een klein beetje kleiner

Resize: Maak de schijf nu weer maximaal groot, de juiste grote word nu aangeven.



Windows herstel actie

Voor een windows schijf:

Start Partition saver (van boot floppy of UBCD ( F3 dan F9))


Selecteer "Update Windows 2000/XP registry."

Selecteer de disk, de partition, de directorie, waar windows staat.

Selecteer nu de disk, de partition, die een andere schijf letter moet krijgen.

Herhaal de stappen indien nodig.
