HKEY_CURRENT_USER

Description of HKEY_CURRENT_USER Registry Subkeys
The information in this key is built from the HKEY_USERS key during the logon process.
Therefore, all information found in the HKEY_CURRENT_USER key is a copy of the subkey HKEY_USERS\username, where username is the name of the active user (or Default).
AppEvents, Control Panel, InstallLocationsMRU, Keyboard Layout, Network, RemoteAccess, Software.
* http://support.microsoft.com/support/kb/articles/q250/4/04.asp

The HKEY_LOCAL_MACHINE branch in the registry defines all of the following specific information for your local computer:
Drivers, Installed hardware, Port settings,Software configuration.
The following list includes the subkey names and a description of them:
Config, Driver, Enum, Hardware, Network, Security, Software, System

HKEY_CURRENT_CONFIG

Description of the HKEY_CURRENT_CONFIG Registry Key in Windows 98/95
The HKEY_CURRENT_CONFIG registry key is stored in the System.dat file
This key points to the current computer configuration in the collection of configurations stored in the HKEY_LOCAL_MACHINE\Config\000x key, which contains the current hardware configuration (where x refers to the configuration number).
* http://support.microsoft.com/support/kb/articles/q250/5/65.asp

HKEY_DYN_DATA

Description of the HKEY_DYN_DATA Registry Key in Windows 95/98, and Win98 SE
Some configuration information in Windows must be stored in RAM because it requires fast modification and retrieval that cannot wait for the registry to send it to the hard disk. You can find all this data in the HKEY_DYN_DATA registry key. The information in this key is newly created every time Windows starts.
* http://support.microsoft.com/support/kb/articles/q250/5/68.asp

BackUp and Restore your Windows 95 Registry.
The first file is regbkup.bat which will back up your files SYSTEM.DAT and USER.DAT to a folder (directory) named REGSAVE you must create before you start.
The second file regrstor.bat will restore both files to the Windows directory. It is assumed that Windows is installed in C:\Windows
@REM regbkup.bat
C:
cd\windows
attrib -h -r -s system.dat
attrib -h -r -s user.dat
copy system.dat c:\regsave\system.bk1
copy user.dat c:\regsave\user.bk1
attrib +r +h +s system.dat
attrib +r +h +s user.dat
copy system.ini c:\regsave
copy win.ini c:\regsave
copy c:\autoexec.bat c:\regsave
copy c:\config.sys c:\regsave

To restore the registry backups from DOS use this one:


REM	regrstor.bat
REM	!!!  This will replace the registry with backups !!!
REM	To cancel use CTRL+C
Pause
C:
cd\windows
attrib -h -r -s system.dat
attrib -h -r -s user.dat
ren system.dat system.xxx
ren user.dat user.xxx
copy C:\regsave\system.bk1 C:\windows\system.dat
copy C:\regsave\user.bk1 C:\windows\user.dat
attrib +r +h +s system.dat
attrib +r +h +s user.dat
REM	you can add the other system files here, some of them may
REM	get changed also when you install programs.
