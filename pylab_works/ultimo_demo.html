Demo Ultimo Mobile



Inloggen


Als eerste moet worden ingelogd op Ultimo, hetgeen gebeurd met het eigen Z-nummer en het daarbij behorende password.


Het laatst ingevoerde Z-nummer wordt onthouden en het scherm start direct met het focus op het password. Ook wordt het toetsenbord automatisch getoond.


Indien een foute combinatie van Z-nummer en password wordt opgegeven, blijft het Z-nummer staan en wordt het password gepoetst. Het focus wordt daarom ook weer op het password gezet.

Plan Lijst


De plan lijst bevat een lijst met geplande aktiviteiten rondom apparatuur. De lijst is gesorteerd op prioriteit. Zodra er uren op een apparaat worden geboekt, bestaat de mogelijkheid de activiteit rondom dit apparaat gereed te melden, waardoor het apparaat uit de plan lijst verdwijnt.


Terug: terug naar hoofd keuzemenu

Weekstaat


Geeft een overzicht van de reeds ingevulde uren voor de geselecteerde week en berekent de subtotalen per dag en per week ( rij onder in de tabel),


De tabel heeft helaas geen edit mogelijkheden ( waarschijnlijk ook te klein), dus is onderaan de pagina een aparte edit regel opgebouwd. Zodra op een regel in de tabel wordt geklikt, wordt de regel overgehaald naar de speciale edit regel en wordt de regel gewist in de tabel (BUG: werkt in de figuur rechts dus nog niet).


Favorieten kunnen worden geladen vanuit de comboboax ( zie "vergaderen" in de figuur rechts).


Daarnaast kan men in alle projecten zoeken, via de knop "Zoek".


Als de gegevens in de speciale edit regel akkoord is, kan deze met de toets "Add" worden toegevoegd aan de weekstaat.


Terug: terug naar hoofd keuzemenu


Menu: deze toets moet nog verdwijnen



Zoek Project


Toont de volledige projectlijst, met een beperkt aantal velden. Boven ieder veld is een edit box aangebracht, waarin de zoekcriteria kunnen worden aangegeven. Bij het zoeken geldt, voor ieder veld wordt gezocht naar "bevat", waarbij niet wordt gelet op de case van de zoektermen.


Zoek: start de zoekactie en toon de projecten die aan de gestelde criteria voldoet


Bekijk: bekijk de details van het project


Select: Keer terug naar waar men vandaan kwam met als argument het geselecteerde project.


Terug: terug naar waar vanuit de zoek funktie was gestart, zonder een geselecteerd project door te geven.


Menu: deze toets moet nog verdwijnen



Apparaat

Lev, leveranciers gegevens inclusief verwijzing website leverancier

Eig, gegevens over de eigenaar van het apparaat

His, historie van het apparaat

Fin, financiële gegevens van het apparaat

Note, algemen notities omtrent het apparaat
