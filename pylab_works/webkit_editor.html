wxKit   (December 2010)


Introduction

wxKit is a wysiwyg html editor with a number of special features. Because at this time wx.WebKit is not yet available, we switched to QTWebKit, embedded in wxPython with the help of PySide and execnet. More details are at the end of this page.The most important features of the editor are :

  • Parts of the text can be indicated as being Python code. The code can be executed in Python and the results (or errors) are automatically fetched within the html document (similar to Sage). Results can be print-output or graphical output products.
  • If code parts can not be evaluated in Python, and if MatLab is installed, that code is executed in MatLab. This gives an unique opportunity to compare Python and MatLab and even gradually move from MatLab to Python (or the other way around ;-) .Also the complete MatLab output (text and graphics) can be caught by the html document. Therefor you can use the MatLab graphics, which are still a bit more flexible and easier to handle than MatPlotLib.
  • With the image editor, you can add rectangles / arrows and texts to images. As all the individual elements are saved in the html page (invisible), you can easily change modified pictures later on.
  • Screen Capture gives you the opportunity to catch any rectangle portion as an embedded picture.
  • With Formula editor you can add Latex formulas as images. The formula is also stored (invisible) in the html document, so it can easily be modified at a later time.
  • You can use either classical styling (use bold / font etc just when you like it), or you can use of three (self definable) style sets. In the latter case it's easy to change the style of all the documents at once.

The button at the upper right, indicates the current color and line width for new elements (not necessarily the properties of the selected elements). The font size is derived from the line width. Clicking on this button, will change the properties of all the selected objects according to this settings.


More than 1 element can be selected, by clicking on objects, while holding the shift key.


Text can be modified by selecting one and only one text object, the current text will appear in the edit box. After modifying the text in the edit box, press Enter and the text of the object will be modified accordingly.
