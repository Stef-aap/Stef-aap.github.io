Patient File

Met behulp van de unit pat_file.pas kunnen en moeten alle acties op een patient file worden uitgevoerd.
Oude patientbestanden worden altijd vertaald, dit is gerealiseerd door altijd gebruik te maken van dezelfde openfile routine, voordat een bestand inhoudelijk echt wordt gelezen.

Public Routines 
get_patient_name_etc Leest alle vaste patientgegevens uit patientfile
get_patient_ntest Leest aantal testregels
get_patient_test Leest de opgegeven testregel
patient_line_to_struct
patient_struct_to_line
