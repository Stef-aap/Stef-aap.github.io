Instellingen venster

Enable Opname

Omschrijving

Door middel van een ingangssignaal de opname automatisch starten en stoppen.

  

Inputs

Parameters

Outputs

Enable

A (ondergrens)

B (bovengrens)

Opname loopt

Gebruik

Opname loopt als A < ENABLE <= B

Een typisch gebruik van deze timer is het verrichten van een aantal verschillende testen, waarbij de testen telkens handmatig gestart worden door het activeren van de juiste marker, en waarbij automatische de opname gestart en gepauzeerd wordt, zoals bijvoorbeeld gewenst bij de ‘autonome neuropathie" test.

Een van de tests zou er als volgt uit kunnen zien: 2 minuten rust, 1 minuut activiteit, 5 minuten rusten, waarbij men geinteresseerd (dus opname) is in de eerste 2+1 minuten. Dit geheel kan gerealiseerd worden met het volgende rekenkanaal:

system signaal marker_2 geeft de start van een marker aan

functie timer2 t1=2 min, t3=1 min, t3=5min, upcounting

functie opname enable A=0, B=2

Zodra nu op marker-2 gedrukt wordt (deze moet uiteraard wel geenabled zijn) start de timer die direct de opname start. Na 3 minuten stopt de timer de opname weer.
