Multi Timer 2

Omschrijving

Met deze funktie kan een tijd of een serie tijden worden afgeteld. De teleenheid bedraagt 1 seconde.

Nadat de eerste teller is afgelopen, start de tweede teller etc. Tellers met een een ingestelde waarde van 0 doen niet mee, derhalve moet de eerste teller ongelijk nul zijn. De aktieve teller wordt aangegeven in het uitgangssignaal mode, zodat dit zichtbaar gemaakt kan worden.

De teller is identiek aan rekenfunktie 8, echter nu kan de timercyclus getriggerd worden doordat het ingangssignaal een drempel overschreidt. Als de timer eenmaal getriggerd is, doorloopt de timer 1 en niet meer dan 1 volledige cyclus.

Bijzonderheden3 in serie geschakelde timers (eenvoudig uitbreidbaar in de toekomst)

 up of down counting

 triggered (one-shot) als ingangssignaal boven ingestelde drempelInputs

1=Trigger als TRIGGER groter dan drempel wordt de timercyclus gestart

Outputs

1=Mode de mode 0=stopped, 1=timer1, 2=timer2, etc

2=Counter teller uitgang

Parameters

drempel de grens waarboven de timercyclus gestart wordt

down/up counting true=down counting

timer 1 preset waarde van timer 1

timer 2 preset waarde van timer 2

timer 3 preset waarde van timer 3

Algoritme

De eerste keer is de tellerwaarde voor start van de teller anders dan de volgende keren. Dit is bewust gedaan om te kunnen constateren dat de timer minstens 1 maal is afgelopen.

Bij upcounting is de startwaarde de EERSTE keer 0, bij een 3 seconde teller verloopt de telling dan als volgt:

voor start =0

eerste seconde =0

tweede seconde =1

derde seconde =2

vierde seconde =3 bij het begin van de vierde seconde is de ingestelde tijd bereikt

De tweede keer verloopt de telling identiek

voor start =3 (LET OP, dit is anders dan de eerste keer)

eerste seconde =0

etc.

Voor downcounting geldt hetzelfde, waarbij de eerste keer de waarde voor start gelijk is aan de af te tellen waarde, bij een volgende keer is de waarde voor start gelijk aan 0

Gebruik

Een typisch gebruik van deze timer is het verrichten van een aantal verschillende testen, waarbij de testen telkens handmatig gestart worden door het activeren van de juiste marker, en waarbij automatische de opname gestart en gepauzeerd wordt, zoals bijvoorbeeld gewenst bij de ‘autonome neuropathie" test.

  

  

Marker1 start Timer1

Timer1 enabled Opname (eventueel slechts in een deel van de timercyclus)

  

  

idem:

Marker2 start Timer2

Timer2 enabled Opname (eventueel slechts in een deel van de timercyclus)

  

  

Op deze wijze worden in 1 opname een aantal testen verzameld, bovendien kan men binnen de opname een test herhalen, waarbij een juist scanning export algoritme de laatste versie voor een berekening kan nemen.

Bovenstaande funkties kunnen met 1 rekenkanaal worden gerealiseerd:

system signaal marker_2

functie timer2

functie opname enable
