Description

This function performs a Fourier Analysis of the input signal.

The number of points to be analysed, the window function and the length of zero-padding can be selected.

Power or magnitude can be selected as output (both lineair or logarithmic) together with the phase information.

In the 32-bit version also real and imaginairy parts can be denoted as outputs.

With a second input signal it's possible to select certain time-samples to form together a time-sequence. If for instance you want to analyse the frequency contents of the hartrate (not of the ECG), you would set the first input signal to the RR-interval and the second input signal to the R-top detector.

This function uses the Intel Signal Processing Library, yielding optimal numerical performance in the case of Pentium and higher processors.

IMPORTANT: the Intel Signal Processing Library (NSP) must be correctly installed.

At the moment the function is somewhat restricted, due to the limitations of the 16-bit environment. In the 32-bit version it will be expanded to a full Fourier Analysis Package. (longer time-sequences, more windows, real/imaginairy parts).

For a quick real-time analysis of sampled data or selection of FFT-parameters, you can select an auto-popup form to control the behaviour of the FFT calculations on-line.

An extra output is available, which is zero if there's not (yet) an FFT available and which identifies the sample number if there is valid FFT-output.

  

Window Blackman

The general Blackman window is given by the next formula (not implemented in 16-bit version)

    w(n) = (a+1) / 2 - 0.5*cos(2*p*n / (N-1)) - (a / 2)*cos(4*p*n / (N-1))

Window Blackman Standard

The Blackman standard window is definied bij a = - 0.16

Window Blackman Optional

provides a modified window that has a 30-dB/octave roll-off, by using

    a  = - sqr [ sin(p / (N-1)) / sin(2*p /(N-1)) ]
