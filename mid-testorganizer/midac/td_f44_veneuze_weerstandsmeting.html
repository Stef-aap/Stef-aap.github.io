Met de Veneuze Weerstandsmeting wordt de weerstand bepaald die het bloed ondervindt in het veneuze systeem in één van de extremiteiten, zie verder achtergrond.

  

Deze functie maakt van een aantal speciale features in Midac gebruik:

  • Eigen popup voor de besturing tijdens opname (besturings panel van Midac wordt uitgezet in de protocol file)
  • Besturings popup verzorgt het starten en stoppen van de opname, zodat enkel de noodzakelijke stukken worden opgeslagen in het opname bestand. Ook worden automatisch markers toegevoegd zodat het analysedeel geheel automatisch kan werken.
  • De grafische x-t vensters worden gecleard en starten links op het scherm bij de start van ieder nieuw meetpunt
  • Het besturings popup bevat ook alle relevante extra testgegevens
  • De file met extra testgegevens wordt ook gebruikt voor het opbergen van de (gewijzigde) analyse gegevens.
  • Focus wordt zoveel mogelijk gehouden op de Start-knop in het popup
  • Ook tijdens de analyse, wordt midac geheel "weggedrukt" door een eigen popup scherm.
  • De functie heeft een interne rapport generator
  • De functie werkt geheel verschillend in opname en achteraf bekijken

 

Beveiliging 

De parameters, welke naar de Phytec controler worden gestuurd, worden begrensd op een maximum. Zodoende, gaat er ook bij een foute calibratie van het druksignaal niets ernstigs mis.

De onderstaande maxima zijn ingesteld:

  • druk max = 1640 ADC = 200 mmHg
  • groffijn max = 200 ADC = 30 mmHg
  • drukok max = 50 ADC = 7 mmHg

 

Opname maken 

Belangrijk element hierin vormt het eigen besturings form: funk44pu.pas.

Dit popup form neemt de gehele besturing over. Zelf doet het form niet veel meer dan een aantal booleans zetten, welke door de execute method van de funktie funk44.pas worden omgezet in acties. Wel verzorgt het popup form het inlezen en wegschrijven van de (gewijzigde) extra testgegevens.

De execute method bestaat grotendeels uit 2 ineengevlochten state-machines, welke op basis van de booleans gezet door het bedienen van de toetsen in het popup form worden gestuurd.

  • state machine voor het verwerken van een noodstop (states > 100)
  • state machine voor de besturing tijdens een geldig meetpunt (states < 100)

Er zijn slechts 2 routines die commando's via de seriële poort naar de meet- en regel-electronika sturen:

    plet_balanceer

    plet_set_druk

Extra testgegevens 

101    de toestand van alle checkboxes

102    de hellings-correctie factoren van links

103    de druk-correctie factor

102    de hellings-correctie factoren van rechts

  

Ingangs Signalen 

1 = plethysmograaf rechts

2 = plethysmograaf links

3 = druk in manchetten

  

Uitgangs Signalen 

1 = afgeleide plethysmograaf rechts

2 = afgeleide plethysmograaf links

De plethysmograaf signalen worden gefilterd teneinde een betrouwbare afgeleide te bepalen.

De afgeleide worden als uitgangs signaal teruggegeven.

  

Nog te doen 

- printen van alle parameters + opmaak van het frf-model

- alle rekenfuncties in de aparte module checken op mogelijke exception errors
