MIDORG  Design & Test
  
Module Acties

Module Acties 
    - Edit Test Protocol wijzigen (of aanmaken van een nieuwe)
    - Opnemen / Afnemen van een test
    - Bekijken van een gemaakte opname / afgenomen test
    - Rapport maken van een gemaakte opname / afgenomen test
    - Import van test-protocollen? / gemaakte opnamen? door een ander testprogramma
    - Converteren van oude testprotocollen
    - Exporteren van de gemaakte opnamen naar een ander programma
    - Instellen van de specifieke parameters van het programma
 

Actie Edit 
Test Protocol wijzigen (of aanmaken van een nieuwe)
De volgende standaarden zijn gedefinieerd:
- wysiwyg editor
- edit actie vindt plaats in een kopie van het opgegeven bestand
- bij het sluiten van het form wordt een (zo goed mogelijk ingevulde) savedialog gebruikt voor het al dan niet opslaan van de wijzigingen.
- indien geen testprotocol wordt opgegeven, wordt een nieuw testprotocol aangemaakt
 

Actie Opnemen 
- als er een databestand is aangemaakt, terugmelding via parameters (result, filedate, filetime) in transport bestand
- als lock_windows true, moet de module de task-switching van windows geheel blokkeren
 

Actie Bekijken 
Hiermee kan een reeds gemaakte opname worden bekeken, in exact dezelfde vorm als tijdens opname.
In het algemeen wordt het niet toegestaan om eenmaal gemaakte opnamen te wijzigen, een module kan hierop een uitzondering maken.
 

Actie Rapport 
Schrijft de resultaten van de gewenste opname in het opgegeven bestand (report file). Als report file geen expliciet path bevat, wordt het programmapath gebruikt. Als uberhaupt geen filenaam is opgegeven, wordt rapport.txt gebruikt.
Als extra parameter wordt hier het gewenste rapport-nivo opgegeven.
Teruggegeven wordt de vorm waarin het rapport is gegenereerd (voorlopig type 1).
    0 = geen rapport
    1 = plain ASCII
    2 = coded ASCII
    3 = quick report ..
MIDORG zorgt voor het samenvoegen van de rapporten van de individuele testen tot een totaal patienten rapport.
 

Actie Import 
Het importeren van databestanden gemaakt met andere programma's.
 

Actie Convert 
Indien van een bepaalde module een dermate grote wijziging ondergaat zodat het noodzakelijk is om een apart vertaalprogramma te gebruiken, teneinde niet alle balst in de nieuwe module mee te nemen, moet een nieuwe bestands-extensie worden aangemaakt.
Indien het programma MidOrg oude bestanden aantreft zal het de actie Converteren aanroepen, waardoor de module zorgt draagt dat alle oude bestanden worden vertaald.
 

Actie Export 
SPSS uitvoer, nog te definieren
De uitvoer is in ieder geval een tab-delimted file, waarbij de eerste kolommen vast gedefinieerd zijn, zodat het hoofdprogramma in de toekomst export-bestanden van meerdere testprotocollen kan samenvoegen.
 

Actie Instellingen 
Wijzigen van het opgegeven module instellingen bestand. Uiteraard bestaat de mogelijkheid bij het afsluiten de instellingen onder een andere naam op te bergen.
Als de bestandsnaam leeg is, danwel niet bestaat, wordt het standaard module-instellingen bestand <path-van-applicatie>\<application-name>.ini gebruikt.
