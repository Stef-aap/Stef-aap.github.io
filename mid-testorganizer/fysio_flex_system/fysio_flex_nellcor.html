Nellcor Oxi Sensor Interface

Inleiding 
 
De Nellcor Puls-Oxi sensors kunnen via een eenvoudig interface kabeltje worden aangesloten op het Fysio Flex System, met als doel het meten van de hartslag. 
Omdat slechts één van de golflengte van de Puls-Oxi sensor wordt gebruikt kan niet de Oxigenatie (SpO2) worden gemeten.
In de toekomst is het misschien mogelijk via een software functie de respiratie te meten.
Voor experimentele studies kan het gewenst zijn de gehele polsgolf in ogenschouw te nemen.
