FysioFlex System
Overzicht

 
Het FysioFlex System vormt samen met het meet- en analyseprogramma MIDAC en een PC een flexibel data-acquisitie en data-analyse systeem voor (fysiologische) signalen. Het FysioFlex System heeft zowel patiënt-veilige (CF) ingangen alsook niet-patiënt-veilige ingangen.
Het FysioFlex System is speciaal ontwikkeld voor gebruik met een notebook (gelijke breedte en diepte), maar kan evengoed op een gewone PC worden aangesloten.

De koppeling naar de PC is standaard uitgevoerd met de 37-polige D-connector (industrie standaard voor dataacquisitie kaarten).

De FysioFlex kan ook geleverd worden met een ingebouwde AD-converter, welke de data via een standaard seriële RS232 poort naar de PC stuurt.

Het Fysio Flex System is de logische opvolger van het standaard DA-frontend, dat gebruikt wordt door nagenoeg alle academische ziekenhuizen in Nederland.

MIDAC is een flexibel, eenvoudig te bedienen (ook door niet-technici) data-acquisitie programma. MIDAC is opgehangen in de MID-TestOrganizer, een patiënten database management systeem.
Voor de MID-TestOrganizer zijn momenteel de volgende modulen beschikbaar: polygrafie, vragenlijsten, spraak-analyse. Momenteel wordt gewerkt aan een module voor neuro-psychologische testen.
