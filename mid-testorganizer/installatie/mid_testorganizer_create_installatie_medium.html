Creatie van een installatie 
Programma (door de toevoeging van de parameter CREATE wordt creatie i.p.v. installatie gestart)
    c:\data\_install\install.exe   create
Gegevens voor de installatie staan in
    <programma directory>\install.ini
oftewel
    c:\data\_install\install.ini
In tegenstelling tot de feitelijke installatie worden bestanden die overschreven dreigen te raken niet hernoemd, maar gewoon overschreven.
 

Het configuratie bestand INSTALL.INI 
Als eerste worden alle te definiëren modules genoemd, de module-naam vormt tevens de section-name, waarin de specifieke gegevens van de betreffende module staan

[Modules]
aantal=10
module 1=MID TestOrganizer
module 2=Battery
module 3=Vraaglst
module 4=Actilog
module 5=NeuroPsycho
module 6=PolyGraf
module 7=PolyScan
module 8=SpraakAnalyse
module 9=Tabellen
module 10=Excel
Vervolgens komen de afzonderlijke gegevens van iedere module
Basepath (waar de programma's komen te staan): hier ontbreekt enkel de driveletter, vandaar dat deze met een dubbele-punt begint
Protocolpath (waar de testprotocollen komen te staan) is enkel een subpath dat wordt toegevoegd aan het basepath
Datapath (waar de testopnamen en patientgegevens komen te staan) eveneens enkel een subpath dat wordt toegevoegd aan het basepath.
Enable geeft de default aan bij de creatie van de module (false=geen installatie creëren)
Inifile (komt alleen voor bij het hoofdprogramma) geeft de default inifile aan, welke bij installatie ook de ingestelde waarden meekrijgt, zodat direct gestart kan worden zonder instellingen te wijzigen.

Source_aantal, het aantal programma source-regels dat volgt
Source1, programma bestanden die moeten worden gekopiëerd (mag normale dos-wildcards bevatten)

Protocol_aantal, het aantal testprotocol source-regels dat volgt
Protocol1, testprotocol bestanden die moeten worden gekopiëerd (mag normale dos-wildcards bevatten)

Hierna komen allerlei bestanden, welke worden ondergebracht in prog / prot / data directory
Vb_prog_aantal, het aantal voorbeeld regels dat volgt
Vb_prog1, voorbeeld bestanden die moeten worden gekopiëerd (mag normale dos-wildcards bevatten) naar de programma directory.
Vb_prog_dest1, subpath, toegevoegd aan het algemene programma-path, waar naar toe bovenstaande bestanden worden gekopieerd.

idem te kopieren naar protocol directory
Vb_prot_aantal
Vb_prot1
Vb_prot_dest1

idem te kopieren naar data directory
Vb_data_aantal
Vb_data1
Vb_data_dest1
