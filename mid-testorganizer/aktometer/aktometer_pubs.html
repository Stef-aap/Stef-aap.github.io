Aktometer type 1,2

Each aktometer is calibrated, by comparing it with a reference device. The calibration results in a 4 point piecewise linear approximation which is automatically used by the recording software to compensate the measurement values.


Aktometer type 3

This aktometer uses a 3-axis, solid state, absolute encoding accelerometer, which has guaranteed accuracy during it's whole lifecycle, so there's no need for further calibration.
