Na klikken op de Help-knop of Help-menu in de TestOrganizer, verschijnt de foutmelding"10093: Successful WSASetup not yet performed".

Oplossing: de helpbestanden zijn van het HTM-type en weergave daarvan vereist dat minimale netwerk-onderdelen geïnstalleerd moeten zijn. 

Dat kan het makkelijkst door in Windows met behulp van de wizard Internet-verbinding een niet bestaande verbinding aan te maken. 

Start de Wizard Internet-verbinding. De hier beschreven procedure geldt voor Windows98. Voor Windows95 en WindowsME verloopt de procedure op een vergelijkbare wijze. 

Kies de optie: "Ik wil mijn bestaande Internet-account overbengen naar deze computer. (Mijn telefoonlijn is verbonden met mijn modem.)" 

Windows98 gaat nu een aantal benodigde bestanden installeren en vraagt mogelijk om de Windows98-CDROM. Wanneer Windows vraagt of bestaande nieuwere bestanden moeten worden behouden, altijd Ja kiezen. Hierna start Windows de PC opnieuw op. 

Na opstarten vervolgt de wizard "Verbinding maken met het Internet". Klik op de kop Annuleren. Er verschijnt een venster met de tekst: "De wizard Internet-verbinding heeft het installeren van uw internet-verbinding niet voltooid. Weet u zeker dat u de wizard wilt afsluiten ?" Zet een vinkje in het vak "De wizard Internet-verbinding voortaan niet meer weergeven" om te voorkomen dat de Wizard telkens na opstarten van de PC opkomt en klik op Ok. 

De Help-functie in TestOrganizer en -modules alsmede Externe Pacemakertester kan nu worden gebruikt. 

Kan ik standaard de letters in het help pagina's verkleinen ?

Ja, er wordt gebruik gemaakt van de standaard instellingen van de microsoft explorer (welke standaard een vreemde instelling heeft).

Start de microsoft explorer

    View

        Internet Options

            font size = small (instellen)

            proportional font = Arial (instellen)

                set as default (aanklikken)

Kan het programma op een netwerk draaien ?

Nee het programma zelf moet altijd op een lokale harde schijf draaien.

Echter voor de data wordt een volledige netwerkondersteuning gegeven, dus op 1 PC opname maken en diezelfde data op een willekeurige andere (netwerk) PC bekijken en analyseren.

Kun je data op een CD-ROM bewaren en direct op deze CD-ROM data bekijken en analyzeren ?

Ja met de windows-95 versie kan dat, mits de CD-ROM binnnen de MID-TestOrganizer wordt ingesteld als netwerk-drive en de juiste netwerkinstellingen worden gebruikt.

Met een CD-ROM zijn er twee problemen, er kan niet naar geschreven worden en de meeste CD-branders zetten alle bestanden op ReadOnly. De MID-TestOrganizer moet kunnen schrijven in zijn data-directory en in principe ook op de netwerk-drive.

Belangrijk, maak een nieuwe datadirectory op de lokale schijf.

Start de testorganizer met een nieuwe ini-file: bijvoorbeeld

    MID_ORG.EXE  cdrom.ini

Nu met de instellingen naar de nieuwe data-directory verwijzen.

De netwerk-parameters dienen in de MID-TestOrganizer als volgt te worden ingesteld:

    update aan het begin = alleen patientdata

    update tijdens het programma = alle data

    update aan het einde = alle data

    netwerk path = path van de CD-ROM, totaan waar de subdirectory \PATIENT\ staat

    computer naam = willekeurige tekst

    deze computer mag enkel data bekijken = JA

    deze computer bevat opnamen data = NEE

    verwijderde bestanden bewaren in windows prullenbak = JA

    readonly flags van bestanden verwijderen = JA
