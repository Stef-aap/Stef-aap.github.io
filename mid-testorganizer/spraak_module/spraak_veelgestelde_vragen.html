Hoe krijg ik accent tekens ? 
Dit wordt niet door het programma bepaald, maar wordt geheel door de instellingen in Windows bepaald. Deze windows instelling is als volgt te bereiken:

Deze Computer
Configuratiescherm
Toetsenbord
Taal
Eigenschappen
Indeling = ...
Raadpleeg de windows handleiding voor uitgebreide informatie. In Nederland zijn twee veel gebruikte indelingen
 - Nederlands  =  accenten verschijnen zodra ingetikt
 - VS (internationaal)  =  accenten worden zo mogelijk gecombineerd met letters (dit werkt meestal het prettigst)
Opmerking desgewenst kunnen meerdere instellingen worden gekozen, waarbij het instellen op een snelle manier via sneltoets of taakbalk kunnen geschieden.
