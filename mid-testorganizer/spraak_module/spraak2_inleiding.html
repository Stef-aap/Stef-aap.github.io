Beloning





Windows Executable

Ieder windows programma kan als beloning worden gebruikt.

Als deze beloning wordt gepresenteerd, wordt het beloningsprogramma gestart in de full-screen mode. Tijdens het beloningsprogramma wordt het spraakprogramma tijdelijk gepauzeerd, totdat het beloningsprogramma is afgesloten.


File Associate

Ieder bestand waarvan op de computer bekend is met welk programma het moet worden geopend, bijvoorbeeld een word document (*.doc) kan als beloning worden gebruikt.

Hiervoor gelden dezelfde regels als voor de windows executable.
